# browser_file_tools

**Links:** [documentation](https://yal.cc/docs/gm/browser_file_tools/) Â· [itch page](https://yellowafterlife.itch.io/gamemaker-browser-file-tools)

This extension lets you do a handful of things on HTML5:

* Paste events
* Drag 'n drop events
* File picker dialogs
* File saving dialogs

For handling images, best used with [Allow Data URI](https://yellowafterlife.itch.io/gamemaker-allow-data-uri).

Have fun *!*
